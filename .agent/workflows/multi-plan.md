# /multi-plan - Multi-Agent Decompositon

## 🎯 Goal
Decompose complex, multi-service tasks into independent units for specialized agents.

## 🛠️ Instructions
1.  **Analyze Request**: Identify backend, frontend, database, and security components.
2.  **Delegate**: Assign specific agents (e.g., `backend-specialist`, `frontend-specialist`) to each unit.
3.  **Set Dependencies**: Define the order of execution (e.g., DB schema before API implementation).
4.  **Synthesize**: Create a master `implementation_plan.md` orchestrating all agents.

## 🏁 Workflow
1.  **Breakdown**: List all technical sub-tasks.
2.  **Assign**: Route to appropriate personas.
3.  **Map**: Create the dependency graph.
