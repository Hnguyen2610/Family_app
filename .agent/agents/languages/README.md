# Language Specialist Reviewers

## Overview
These agents provide deep expertise in specific programming languages, focusing on idioms, performance, and build-time safety.

## Agents
- **rust-reviewer**: Expert in memory safety, ownership, and Tokio/Axum patterns.
- **go-reviewer**: Expert in concurrency (goroutines/channels), interfaces, and gRPC.
- **swift-reviewer**: Expert in Actors, protocol-oriented programming, and iOS persistence.
- **java-reviewer**: Expert in Spring Boot, Maven/Gradle, and JPA optimization.
- **php-reviewer**: Expert in Laravel, security best practices, and modern composer patterns.

## Instructions
Invoke these agents when working on non-JS/TS modules or when cross-language integration is required.
